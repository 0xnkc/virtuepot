Honeyd - Script Contributions

Here you will find useful scripts contributed by other people to run with Honeyd.

The scripts are provided without warranty by their authors. I have not tested them for their usability.

Script Purpose Author Date
telnet-emul Telnet Simulation for Linux, Solaris and Windows Daniel Cid 2003-06-20
Honeyd Regression Testing Scripts to check Honeyd personalities Eric Thomas 2003-05-19
Honeyd Scripts Several scripts for Suse and Windows Fabian Bieker 2003-04-10
HoneydWeb-0.4 web server that speaks HTTP/0.9 - HTTP/1.1 Kevin Timm 2003-02-15
POP.emulator Emulates a POP3 server with a configurable user base. Rob MacGregor 2002-10-25
iisemul8 Web server perl script which is indistinguishable from Microsoft's IIS product at a topical level. Rain Forest Puppy & HDM 2002-07-25
ftp.sh FTP (WU-FTPD) Honeypot-Script simulates a few functions of a old WU-FTPD 2.6.0 Maik Ellinger 2002-07-24
smtp.sh SMTP (Sendmail) Honeypot-Script simulates a few functions of a Sendmail 8.12.2 Server Maik Ellinger 2002-07-24
pop3.sh POP3 Honeypot-Script simulates a few functions of a QPOP 2.53 Maik Ellinger 2002-07-24
